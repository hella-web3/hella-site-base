## ğŸ“¦ Installation

Config and setup.

**Install rbenv**
```terminaloutput
brew install rbenv
rbenv init
```

**Install ruby**
```terminaloutput
rbenv install 3.4.3
rbenv global 3.4.3
```

**Update bundler**
```terminaloutput
gem install bundler -v 2.6.7
```
